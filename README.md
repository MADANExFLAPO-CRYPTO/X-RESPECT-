# X-RESPECT Clan Website

A visually engaging HTML, CSS, and JavaScript website for the **X-RESPECT Clan**, featuring:

- Interactive **welcome pages**  
- **Password-protected entry** with a math challenge  
- **Social media links**: Discord, WhatsApp, Instagram  
- Background **video and music**  
- Developer page accessible after secondary welcome  

## Features

- Modern, responsive layout  
- Hover animations on buttons and social icons  
- Music toggle button  
- Easy to update social media links  

## How to Use

1. Open `index.html` in your browser to view locally.  
2. Upload all files to a web host (GitHub Pages, Netlify, Vercel, etc.) to make it live.  
3. Make sure background video and music URLs are accessible online.
